# Statistics-For-Data-Science-Project-Youtube-Video-Analysis
# YouTube Trending Video Analytics

## Project Overview

This project analyzes trending YouTube videos in India using statistical techniques and data visualization. The objective is to explore patterns in video performance, audience engagement, and content popularity using descriptive statistics and exploratory data analysis (EDA).

The analysis focuses on metrics such as views, likes, dislikes, and comment counts to identify trends and understand user engagement on the YouTube platform.

---

## Objectives

* Perform exploratory data analysis on YouTube trending video data.
* Calculate descriptive statistical measures.
* Visualize the distribution of views, likes, and comments.
* Identify relationships between engagement metrics.
* Detect outliers and unusual patterns in the dataset.
* Draw meaningful insights from trending video performance.

---

## Dataset

The dataset contains information about trending YouTube videos in India, including:

* Video Title
* Channel Name
* Category ID
* Views
* Likes
* Dislikes
* Comment Count
* Publish Time
* Trending Date

Dataset Source: Kaggle YouTube Trending Videos Dataset

## Analysis Performed

### Data Cleaning

* Handled missing values
* Removed duplicate records
* Checked data types and dataset structure

### Exploratory Data Analysis

* Distribution of video views
* Analysis of likes and comments
* Top trending channels
* Most viewed categories
* Relationship between views and likes

### Correlation Analysis

A correlation matrix was generated to examine the relationships among:

* Views
* Likes
* Dislikes
* Comment Counts

---

## Key Findings

* Most videos receive relatively low view counts, while a small number of videos achieve extremely high popularity.
* Views and likes show a strong positive correlation.
* Certain channels appear more frequently in the trending section than others.
* Some videos act as significant outliers with exceptionally high engagement.
* Trending content is concentrated within a few popular categories.
